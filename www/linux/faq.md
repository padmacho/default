## How to find Ubuntu version
**lsb_release -a** can be used to find Ubuntu version
```bash
vagrant@python-m1:~$ lsb_release -a
No LSB modules are available.
Distributor ID: Ubuntu
Description:    Ubuntu 16.04.1 LTS
Release:        16.04
Codename:       xenial
```
